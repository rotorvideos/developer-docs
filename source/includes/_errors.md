# Errors

The Rotor Partner API uses the following error codes:

Error Code | Meaning
---------- | -------
400 | Bad Request
401 | Unauthorized
402 | Quota Exceeded
403 | Forbidden
404 | Not Found
405 | Method Not Allowed
406 | Not Acceptable
429 | Too Many Requests
500 | Internal Server Error
503 | Service Unavailable
